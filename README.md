# On-Together Macro Scripts
这个仓库包含为“On-Together”游戏设计的宏脚本。这些脚本使用Python的`pynput`库监听鼠标和键盘事件，实现自动化绘制、连点等功能，帮助玩家在游戏中快速完成某些操作。

## 效果图

![游戏中黑板画图效果图](img.png)

## 文件说明

- **侧键绘制圆形.py**: 使用鼠标侧键绘制圆形。适合在游戏中快速画圆。
- **侧键连点.py**: 使用鼠标侧键触发连点操作。用于需要快速点击的场景。
- **侧键绘制点阵画.py**: 从`dots.txt`文件读取点阵图案数据，使用鼠标侧键绘制点阵画。图案由点（.）和空格组成。
- **绘制数独.py**: 绘制数独网格、粗分隔线和数字。支持从预设谜题填充。
- **dots.txt**: 点阵图案的数据文件，用于`绘制点阵画.py`。

## 依赖

- Python 3.x
- `pynput` 库：用于监听鼠标和键盘事件。

安装依赖：
```bash
pip install pynput
```

## 使用方法

1. 确保Python环境已安装并配置好。
2. 运行相应的脚本（例如：`python 侧键绘制圆形.py`）。
3. 根据脚本提示操作鼠标和键盘。


### 绘制点阵画
- 确保`dots.txt`文件在同一目录。
- 鼠标移动到起点，按侧键开始绘制点阵图案。
